# 📦 Inventory Management System

A full-stack **Inventory Management System** developed using modern web technologies to efficiently manage products, inventory, suppliers, orders, and user purchases. The system provides separate dashboards for **Managers** and **Users** with secure JWT authentication and role-based access.

---

## 🚀 Technologies Used

- React.js
- Node.js
- Express.js
- MongoDB
- Bootstrap
- JWT Authentication
- Axios
- React Router
- React Toastify
- Chart.js

---

## ✨ Features

### 👨‍💼 Manager Module

- Secure Manager Login
- Dashboard with Inventory Statistics
- Product Management (Add, Edit, Delete)
- Category Management
- Supplier Management
- Order Management
- Activity Logs
- Reports & Analytics
- Profile Management

### 👤 User Module

- User Registration & Login
- Browse Products
- Shopping Cart
- Checkout System
- Order History
- Profile Management

## 🛠️ Installation

### Clone Repository

```bash
git clone https://github.com/khushidwi/Inventory-Management-System.git
```

### Install Backend

```bash
npm install
npm run dev
```

### Install Frontend

```bash
cd client
npm install
npm run dev
```

---

## ⚙️ Environment Variables

Create a `.env` file inside the **server** folder.

```env
PORT=5000
MONGODB_URI=mongodb_connection_string
JWT_SECRET=secret_key
EMAIL_USER=email@gmail.com
EMAIL_PASS=app_password
```

# Inventory Management System

A full-stack Inventory Management System built using:

- React.js
- Node.js
- Express.js
- MongoDB
- Bootstrap
- JWT Authentication

## Features

### Manager
- Dashboard
- Product Management
- Categories
- Suppliers
- Orders
- Activity Logs
- Reports
- Profile Management

### User
- Register/Login
- Browse Products
- Shopping Cart
- Checkout
- Order History
- Profile Management

## Installation

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```

Create a `.env` file inside the `server` folder with:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
EMAIL_USER=your_email@gmail.com
EMAIL_PASS=your_app_password
```
